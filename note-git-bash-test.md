```shell
$ pwd puis Entrée
```

```shell
$ ls puis Entrée &ldsh;
```          

## ✔ Commande : ls -ah (bash espace NO&ldsh; HTML ENTITY <br/>Essai HTML CODE &#8626;)


## ✔ Commande : rm delete (shell)
```shell

```

```shell
$ rm -d nom_du_fichier puis Entrée

```
